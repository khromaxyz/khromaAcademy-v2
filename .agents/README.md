# Agentes de Prompt Engineering

Esta pasta contém agentes especializados para diferentes tipos de melhorias e transformações de prompts.

## Estrutura

### prompt-improvement/
Contém agentes especializados em melhorar prompts de diferentes formas:

- **corrigir-prompt.md** - Corrige apenas erros de caligrafia, organização e ambiguidade, mantendo a estrutura original
- **melhorar-paragrafo.md** - Melhora o prompt e condensa em um parágrafo conciso
- **criar-prompt-completo.md** - Cria um prompt completo usando melhores práticas (papel, contexto, instruções, etc.)
- **criar-variacoes.md** - Gera múltiplas variações do mesmo prompt, útil quando o usuário não sabe exatamente o que quer

## Como Usar

1. Selecione o agente apropriado para sua necessidade
2. Copie o prompt do agente
3. Cole no chat com o prompt que deseja melhorar
4. O agente processará e retornará o resultado

## Exemplos de Uso

- **Corrigir**: "Preciso que você corrija apenas os erros de digitação e deixe mais claro"
- **Parágrafo**: "Transforme este prompt em um parágrafo bem escrito e conciso"
- **Completo**: "Crie um prompt completo e profissional seguindo as melhores práticas"
- **Variações**: "Crie 3-5 variações diferentes deste prompt para eu escolher"
