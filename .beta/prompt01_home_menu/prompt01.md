Crie um design de interface web (HTML/CSS) para a seção principal 'Home' de uma plataforma de ensino de Ciência da Computação de elite, utilizando uma estética 'Dark RGB Chroma' de altíssimo nível. O objetivo é criar o melhor design possível, com UX/UI impecável e criatividade ilimitada para impressionar qualquer usuário. **Seja extremamente criativo e ouse em soluções visuais que deixem o usuário boquiaberto.**

O layout deve focar exclusivamente no conteúdo central:

Hero Section: Tipográfica, impactante e limpa.
Cards de Disciplina: Design minimalista e sofisticado, apresentando bordas dinâmicas com efeito 'Chroma RGB' (gradiente arco-íris animado) que percorre o contorno de forma fluida e elegante.
Requisitos Visuais:

Estética: Dark mode premium, limpo e organizado.
Efeitos: Use animações suaves, micro-interações refinadas e o efeito RGB Chroma dinâmico nas bordas como destaque principal, mas sem poluição visual.
Foco: Elegância, legibilidade e apresentação "state-of-the-art" dos cursos.
Escopo: Ignore cabeçalho e barra lateral. Foque apenas na Home.
Sinta-se livre para escrever quantas linhas de código forem necessárias para atingir um resultado visualmente deslumbrante e funcionalmente perfeito. Use CSS moderno (Grid, Flexbox, variáveis, keyframes) para criar uma experiência imersiva e única.
