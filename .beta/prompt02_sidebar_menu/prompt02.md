Crie um design de interface web (HTML/CSS) para a Sidebar de Navegação da plataforma Khroma Academy, elevando o nível para uma estética 'Dark RGB Chroma' de elite. O objetivo é refatorar a sidebar para que ela seja visualmente deslumbrante, moderna e funcional, seguindo o Design System.
Foco Exclusivo: Refatoração da Sidebar (Menu Lateral).
Requisitos Visuais e Comportamentais:
Estética e Materiais:
Glassmorphism Avançado: Utilize backdrop-filter: blur() com transparência ajustada para criar um efeito de vidro fosco profundo e elegante sobre o fundo dark (#0a0a0a).
Bordas Sutis: Use bordas finas e translúcidas (rgba(255, 255, 255, 0.08)) para definir a estrutura sem pesar.
Interatividade RGB Chroma:
Itens de Navegação: Ao passar o mouse (hover) ou quando ativo, o item deve exibir uma Borda RGB Chroma (estilo arco-íris colorido e animado) que percorre o contorno ou a lateral. O efeito não é apenas um neon estático, mas um gradiente multicolorido vibrante que se move, capturando a essência 'Chroma'.
Micro-interações: Ícones e textos devem ter transições suaves de cor e posição. Sinta-se livre para explorar efeitos de "brilho" que seguem o cursor.
Estrutura e Layout:
Topo: Logo da Khroma Academy com presença marcante.
Menu: Lista de links com ícones minimalistas e tipografia limpa (Inter).
Footer: Área de perfil do usuário com avatar e ações rápidas, mantendo a estética glassmorphism.
Responsividade: Considere como a sidebar se comporta em telas menores (ex: recolher para ícones ou menu hambúrguer).
Escopo:
Ignore o conteúdo da página (Home, Dashboard, etc.). Concentre-se apenas na estrutura e estilo da Sidebar.
Instruções Finais:
Escreva todo o HTML e CSS necessário.
Use CSS Variables para as cores RGB Chroma para facilitar a manutenção.
Surpreenda: Crie algo que faça o usuário sentir que está navegando em uma interface do futuro.
