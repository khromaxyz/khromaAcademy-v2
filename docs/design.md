# Design System - Khroma Academy

## Visão Geral

O Khroma Academy utiliza um design system baseado em **dark theme** com acentos **RGB Chroma** vibrantes, criando uma experiência visual moderna e imersiva.

## Paleta de Cores

### Cores Principais (RGB Chroma)
- **Vermelho**: `#FF4141`
- **Verde**: `#41FF41`
- **Azul**: `#4141FF`
- **Magenta**: `#FF41FF`
- **Ciano**: `#41FFFF`
- **Amarelo**: `#F2FF41`

### Background Dark
- **Preto Base**: `#0a0a0a`
- **Superfície**: `#151515`
- **Superfície Elevada**: `#1f1f1f`
- **Bordas**: `rgba(255, 255, 255, 0.08)`

## Design do Chat (Gemini Chatbot)

O chat segue o mesmo padrão visual da plataforma:

- **Background**: Transparente com efeitos de glassmorphism
- **Bordas**: Sutil com glow RGB na lateral esquerda
- **Mensagens**: Background escuro com bordas arredondadas
- **Input**: Campo com blur e bordas sutis
- **Acentos**: Cores RGB Chroma para elementos interativos e highlights

## Princípios de Design

- **Dark First**: Tema escuro como padrão
- **RGB Accents**: Cores vibrantes para destaque e interatividade
- **Glassmorphism**: Efeitos de vidro fosco com blur
- **Minimalismo**: Interface limpa e focada
- **Micro-interações**: Animações suaves em todos os elementos
